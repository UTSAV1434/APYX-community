# APYX Design System

## Purpose

The APYX Design System defines the visual language and reusable UI rules for the entire platform.

Every page and component must follow these guidelines.

---

# Design Principles

- Premium over flashy
- Minimal over cluttered
- Consistency over creativity
- Motion should support usability
- Content should always remain the focus
- Every component should feel connected

---

# Layout

## Grid

- 12-column responsive grid
- Max Content Width: 1280px
- Full Width: 1440px
- Section Padding: 96px desktop / 64px tablet / 32px mobile
- Container Padding: 24px desktop / 20px tablet / 16px mobile

---

# Breakpoints

Mobile: <640px

Tablet: 640px–1024px

Desktop: 1024px–1440px

Large Desktop: >1440px
