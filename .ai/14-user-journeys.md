# APYX User Journeys

Version: 1.0

---

# Purpose

This document defines how every type of user interacts with APYX.

Every page should exist to help users complete a goal with the fewest possible steps.

Navigation should feel effortless.

Every interaction should move users closer to their objective.

# Navigation Principles

Users should reach their goal within three clicks whenever possible.

Primary navigation should remain simple.

Never overwhelm users with options.

---

# Search Experience

Search should be available globally.

Users should quickly find:

- Events
- Announcements
- Resources
- Gallery
- Team Members

---

---

# AI Rules

Before creating or modifying any page:

- Identify the target user.
- Understand their goal.
- Remove unnecessary steps.
- Prioritize clarity.
- Maintain consistency.
- Optimize for speed.
- Preserve APYX identity.

If a page does not help a user complete a meaningful journey, redesign or remove it.
