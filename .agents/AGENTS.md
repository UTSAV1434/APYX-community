# APYX Community UI/UX Rules

<!-- BEGIN:user-functional-lock -->
**UI Lock & Functionality Focus**
- The UI is currently LOCKED. Do not spend time polishing animations, gradients, typography, spacing, or visual effects.
- Prioritize **functionality over appearance** at all times.
- Build the project using reusable React components and a centralized design system so the entire UI can be easily replaced later without affecting functionality.
- The ultimate goal is to complete a production-ready functional platform first.
<!-- END:user-functional-lock -->
